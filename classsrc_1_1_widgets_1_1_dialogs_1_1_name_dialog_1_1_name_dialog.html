Ausführliche Beschreibung

Dialog zur Auswahl der darzustellenden Merits.

Alle Merits darzustellen ist wohl etwas viel. Über diesen Dialog kann der Benutzer auswählen, welche und im Falle von Merits mit Zusatztext, wieviele er angezeigt haben möchte.

Dokumentation der Elementfunktionen

def src.Widgets.Dialogs.NameDialog.NameDialog.getNames (   self)

Liest die Namen des Charakters aus und zeigt sie gleich an.

Besteht ein Namen nur aus Whitespace, wird er nicht beachtet. Leer darf er allerdings sein.

Zu beachten:
Momentan wird nur die echte Identität berücksichtigt, also immer nur die Identität an INdexposition 0.
def src.Widgets.Dialogs.NameDialog.NameDialog.saveNames (   self)

Speichert die eingegebenen Namen im Charakter-Speicher.

Zu beachten:
Derzeit wird nur /eine/, die echte Identität unterstüzt. Also wird immer nur die Identität an Indexposition 0 überschrieben.

Ein Textfeld, das nur aus Whitespace besteht, wird als leer betrachtet.

def src.Widgets.Dialogs.NameDialog.NameDialog.showNames (   self)

Zeigt den resultierenden Namen an.

Einmal der Name, wie er später auf der Schaltfläche zu sehen sein wird, die diesen Dialog aufruft, einmal den vollständigen Namen mit allen Bestandteilen.
