Ausführliche Beschreibung

Darstellung von ankreuzbaren Quadraten.

Ein einfacher ganzzahliger Wert wirden in Form angekreuzter Quadrate dargestellt. Die bis zum Maximalwert übrigen Quadrate sind nicht ausgefüllt.

Wird das Widget disabled, wird der Alphakanal genutzt, um die Quadrate teilweise durchsichtig zu machen und sie so grau erscheinen zu lassen.

int Squares::columnMax [read, write]

Speichert die maximale Anzahl von Quadraten, welche in enie Zeile gezeichnet werden. Alle bis zum maximum überzähligen Quadrate werden in eine neue Zeile gesetzt, welche natürlich wiederum bei Füllung dieser Zahl umgebrochen wird.

Zugriffsfunktionen:

Benachrichtigungssignal:

int Squares::value [read, write]

Speichert den aktuellen Wert des Widgets.

Dieser Wert stellt die Zahl der ausgefüllten Quadrate dar. Die Gesamtzahl der dargestellten Quadrate ist in maximum gespeichert.

Soll value auf einen Wert gesetzt werden, der über dem Maximum liegt, wird er nur auf das Maximum gesetzt, soll er unter das Minimum gesetzt werden, wird er auf Minimum gesetzt.

Zugriffsfunktionen:
value(), setValue()
Benachrichtigungssignal:
