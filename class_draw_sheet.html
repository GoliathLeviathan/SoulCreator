Ausführliche Beschreibung

Führt das Drucken des Charakters aus.

Mit Hilfe dieser Klasse können die Charakterwerte auf Papier gebannt werden.

Dokumentation der Elementfunktionen

void DrawSheet::enforcedTraitLimits ( cv_Trait::Type  type) [signal]

Dieses Signal wird ausgesandt, wann immer ein Eigenschaftstyp nicht auf die vorgegebene Charakterbogen-Matrix paßt.

Warnung:
Aktuell ist das Programm so beschaffen, daß dieses Signal ausgesandt wird der Druck aber fortgesetzt wird, wobei jedoch die überzähligen Eigenschaften ignoriert werden.
Parameter:
typeDieses Argument teilt mit, bei der Bearbeitung welchen Eigenschaftstyps das Limit überschritten wurde.
