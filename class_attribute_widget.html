Ausführliche Beschreibung

Das Widget, in welchem sämtliche Attribute angeordnet sind.

Die Attribute werden in diesem Widget angeordnet.

Noch zu erledigen:
Die Attribute in ein Gridlayout packen, damit bei den Werwölfen die Manipulation nicht so heraussticht.
