Ausführliche Beschreibung

Grundlegender Datentyp für eine Charaktereigenschaft.

Dies ist die Grundlage für sämtliche Charaktereigenschaften.

Dokumentation der Aufzählungstypen

enum cv_AbstractTrait::Category

Welcher Kategorie (mental, physisch, sozial) gehört die Eigenschaft an?

Flag Description
Category::CategoryNo Gehört keiner Kategorie an.
Category::Mental Gehört der mentalen Kategorie an.
Category::Physical Gehört der physischen Kategorie an.
Category::Social Gehört der sozialen Kategorie an.
Category::Item Gegenstände.
Category::FightingStyle Kampfstile.
Category::DebateStyle Debattierstile.
Category::Extraordinary Psychische Phänomene wie Psi oder Hellsicht. Auch mit Geistern in Verbindung stehende Merits.
Category::Mild Milde Geistesstörungen
Category::Severe Ernste Geistesstörungen.
Category::BreedPower Kräfte welche hauptsächlich Angehörigen der jeweiligen Brut zugeschrieben werden.
Category::FactionPower Kräfte welche hauptsächlich Angehörigen der jeweiligen Fraktion zugeschrieben werden.
Category::ClubPower Kräfte welche hauptsächlich Angehörigen der jeweiligen Sonderfraktion (Blutlinie, Legat etc.) zugeschrieben werden.
enum cv_AbstractTrait::Type

Zu welchem Typus gehört die Eigenschaft? Attribut, Fertigkeit, Vorzug etc.

Flag Description
Type::TypeNo Gehört keinem der genannten Typen an.
Type::Virtue Ist eine Tugend.
Type::Vice Ist eie Fertigkeit.
Type::Attribute Ist ein Attribut.
Type::Skill Ist eine Fertigkeit.
Type::Merit Ist ein Vorzug.
Type::Morale Ist die Moraleigenschaft.
Type::Super Ist das Superattribut (Gnosis, Primal Urge, Blood Potency).
Type::Power Ist eine übernatürliche Kraft (Discipline, Arcanum, Renown...).
