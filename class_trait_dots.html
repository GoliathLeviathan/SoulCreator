Ausführliche Beschreibung

Eine Darstellung von Werten in Form ausgefüllter Punkte.

Ein einfacher ganzzahliger Wert wirden in Form ausgefüllter Punkte dargestellt. Die bis zum Maximalwert übrigen Punkte sind nicht ausgefüllt.

Wird das Widget disabled, wird der Alphakanal genutzt, um die Punkte teilweise durchsichtig zu machen und sie so grau erscheinen zu lassen.

Es besteht die Möglichkeit aus der Menge an Werten zwischen minimum und maximum einige zu verbieten.

int TraitDots::value [read, write]

Speichert den aktuellen Wert des Widgets.

Dieser Wert stellt die Zahl der ausgefüllten Punkte dar. Die Gesamtzahl der dargestellten Punkte ist in maximum gespeichert.

Soll value auf einen Wert gesetzt werden, der über dem Maximum liegt, wird er nur auf das Maximum gesetzt, soll er unter das Minimum gesetzt werden, wird er auf Minimum gesetzt.

Soll value auf einen verbotenen Wert gesetzt werden, wird er auf den nächstkleineren, erlaubten Wert gesetzt.

Zugriffsfunktionen:
value(), setValue()
Benachrichtigungssignal:
valueChanged()

void TraitDots::forbidAll ( ) [slot]

Verbiete alle Werte.

Zu beachten:
Es werden alle Werte verboten. Und zwar jeder Einzelne Wert zwischen minimum() und maximum() (einschließlich beider).
Warnung:
Gefährlich, da bei einer Veränderung von minimum() und maximum() die Verbotenen Werte nicht verändert werden.
