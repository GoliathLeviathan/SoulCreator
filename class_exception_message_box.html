Ausführliche Beschreibung

Diese Klasse stellt ein Standardnachrichtenfenster für die Ausnahmebehandlung dar.

Über das Standardnachrichtenfenster können die Nachrichten über Ausnahmen bequem an den Nutzer weitergegeben werden.

static StandardButton exception QWidget * ExceptionMessageBox::parent [static]

Standardisierte Dialogbox für die Mitteilung einer Ausnahme an den Benutzer. Dient bislang Debug-Zwecken und sind noch keine normierten Fehlermeldungen.

Noch zu erledigen:
Den Dialog so umwandeln, der er auch als Fehlermeldung einem Benutzer präsentiert werden kann. Dies ist natürlich nicht einfach nur über diese Klasse zu lössen sondern wird Änderungen in der Exception -Klasse erfordern.
